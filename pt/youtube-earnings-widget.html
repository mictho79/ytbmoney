Perguntas Frequentes

É realmente gratuito?

Sim. A calculadora está hospedada na CDN global da Cloudflare sem custo para você. Não há paywall, requisitos de conta ou limites de uso.

O widget vai quebrar se você atualizar o site?

A URL do iframe (https://ytbearnings.com/) é permanente. Qualquer atualização de design ou dados é aplicada automaticamente — seu widget sempre mostrará a versão mais recente.

O widget usa cookies ou coleta dados?

A calculadora não define nenhum cookie primário. Os logs de acesso padrão da Cloudflare se aplicam. Nenhum dado pessoal é coletado ou armazenado pelo próprio widget.

Posso incorporar uma página de calculadora específica via iframe?

Sim — basta alterar o src para qualquer página em ytbearnings.com, como /youtube-cpm-calculator.html ou /youtube-niche-rpm-calculator.html.
