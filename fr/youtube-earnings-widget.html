Ajoutez un calculateur de revenus YouTube entièrement fonctionnel sur n'importe quel blog ou site web en quelques secondes — gratuit, sans inscription, sans clé API.

Comment Intégrer le Calculateur

Copiez le code iframe ci-dessous et collez-le n'importe où dans votre HTML — dans un article de blog, un widget de barre latérale ou une page d'outils dédiée. Le calculateur se charge directement depuis ytbearnings.com et reste toujours à jour.

Intégration Standard (Recommandée)

Collez ce code là où vous souhaitez afficher le calculateur :

<iframe
  src="https://ytbearnings.com/"
  width="100%"
  height="600"
  frameborder="0"
  title="YouTube Money Calculator"
  loading="lazy">
</iframe>

Questions Fréquemment Posées

Est-ce vraiment gratuit ?

Oui. Le calculateur est hébergé sur le CDN mondial de Cloudflare sans coût pour vous. Pas de paywall, pas d'exigence de compte, pas de limites d'utilisation.

L'intégration sera-t-elle cassée si vous mettez à jour le site ?

L'URL de l'iframe (https://ytbearnings.com/) est permanente. Les mises à jour de design ou de données sont appliquées automatiquement — votre intégration affichera toujours la dernière version.

Le widget utilise-t-il des cookies ou collecte-t-il des données ?

Le calculateur ne définit aucun cookie propriétaire. Les journaux d'accès Cloudflare standard s'appliquent. Aucune donnée personnelle n'est collectée ou stockée par le widget lui-même.

Puis-je intégrer une page de calculateur spécifique en iframe ?

Oui — il suffit de modifier le src pour n'importe quelle page de ytbearnings.com, par exemple /youtube-cpm-calculator.html ou /youtube-niche-rpm-calculator.html.
